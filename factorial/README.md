# فاکتوریل

فاکتوریل به عملی در ریاضیات گفته میشود که تمام اعداد طبیعی قبل از عدد مورد نظر را به همراه خود عدد در یک دیگر ضرب کنیم.

n! = 1 * 2 * 3 * 4 * ... * (n-1) * n

مثال : 

5! = 1 * 2 * 3 * 4 * 5 = 120
